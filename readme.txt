Курьерская служба «Забери-Товар»

Модуль для расчета стоимости доставки и экспорта заказов через API курьерской службы «Забери-Товар».

Особенности:
- Необходим vQmod.
- Работает с модулем simple.

Совместимость:
OpenCart 1.5.6.4
ocStore 1.5.5.1.2
